Pick photos from picasa album and display them randomly

== Installation ==

1. Upload `picasa.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings -> Picasa Photos and set your username
4. Go to Presentation -> Widgets and add the Picasa widget
5. Configure your widget
