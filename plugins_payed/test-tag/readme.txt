== Description ==

Allow to tag as test some content (posts, pages, medias, custom posts, categories, widgets) that you can delete in one click.

This is a nice plugin for developpers who have to add some dummy content to a website and don't want it to appear when the site is online.

Features:

*   Tag as test individualy each post (any type including media and custom), category, comment and widget,
*   Bulk tag as test posts (any type including media and custom), categories and comments,
*   Delete in one click all "Test Tagged" contents,
*   French and english language support.
*   New feature : widget title bar is blue when the widget is a test.

== Installation ==

1. Upload `test-tag.zip` to the `/wp-content/plugins/` directory
2. Unzip archive
3. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==
